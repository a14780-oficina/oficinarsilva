# 🐍 Exercícios de Condicionais em Python

Este repositório reúne vários exercícios introdutórios de **Python**, focados no uso de **estruturas condicionais (`if`, `elif`, `else`)**.  
Os exercícios fazem parte da minha aprendizagem no **Curso Técnico de Gestão e Programação de Sistemas Informáticos (10.º ano)**.

---

## 📘 Conteúdos dos Exercícios

### 🔹 Exercício B1 — Número Positivo ou Negativo
Pede ao utilizador um número e indica se é **positivo** ou **negativo**.

### 🔹 Exercício B2 — Maior de Idade
Recebe a idade do utilizador e informa se é **maior** ou **menor de idade**.

### 🔹 Exercício B3 — Número Par ou Ímpar
Verifica se um número introduzido é **par** ou **ímpar**.

### 🔹 Exercício B4 — Comparação de Dois Números
Compara dois números e indica qual é o **maior**.

### 🔹 Exercício B5 — Password Simples
Compara uma password introduzida com uma password pré-definida e indica se o acesso é **permitido** ou **negado**.

---

## 📗 Exercícios Intermédios

### 🔹 Exercício I1 — Classificação de Nota
Classifica uma nota entre **Excelente**, **Bom**, **Suficiente** ou **Reprovado**.

### 🔹 Exercício I2 — Classificação de Idade
Classifica a idade do utilizador como **criança**, **jovem**, **adulto** ou **sénior**.

### 🔹 Exercício I3 — Múltiplo de 3 e 5
Verifica se um número é múltiplo de **3**, **5**, ou de **ambos**.

### 🔹 Exercício I4 — Login com Utilizador e Password
Compara um username e password com valores pré-definidos e devolve se o login é válido.

### 🔹 Exercício I5 — Número dentro de Intervalo
Verifica se um número está dentro do intervalo **[10, 20]**.

---

## 📙 Exercícios Avançados

### 🔹 Exercício A1 — Sistema Multibanco Simples
Simula um levantamento de dinheiro, verificando se o valor pedido é inferior ao saldo disponível.

### 🔹 Exercício A2 — Maior de Quatro Números
Recebe quatro números e determina qual é o **maior**.

### 🔹 Exercício A3 — Classificação de IMC (Simplificado)
Calcula o **IMC** e classifica o utilizador como:  
**Baixo peso**, **Normal**, **Excesso de peso** ou **Obesidade**.

### 🔹 Exercício A4 — Sistema de Desconto
Aplica um desconto de acordo com o valor da compra:
- ≥ 100 → 10%  
- ≥ 50 → 5%  
- < 50 → sem desconto

### 🔹 Exercício A5 — Verificação de Ano Bissexto
Verifica se um ano é **bissexto** ou **normal**.
